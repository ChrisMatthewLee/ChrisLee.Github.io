Description

Our app is a Snapchat-like app that we created with Android App Inventor where you can take a picture and doodle on it with different colors. You can use the camera to take a picture, pick a color from the color button screen, and change the size of the brush.

The Code

In this bit of code, whenever you touch the canvas, the DrawCircle function is called. This function gets the x and y values, gets the global dot size to determine the radius size for the brush, and creates a fill for the brush stroke.
